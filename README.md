# Curso Práctico de Ecuaciones Diferenciales Ordinarias y Funciones Especiales, Con Aplicaciones

Este repositorio contiene el código fuente en LaTeX del libro **"Curso Práctico de Ecuaciones Diferenciales Ordinarias y Funciones Especiales, Con Aplicaciones"**.

## Estructura del Proyecto

El proyecto está organizado de la siguiente manera:

Libro_EDO_Uasd/ ├── Cuerpo/ │ ├── Capitulos/ │ │ ├── Inicio/ │ │ │ └── inicio.tex │ │ ├── Dedicatoria/ │ │ │ └── Dedicatoria.tex │ │ ├── Agradecimiento/ │ │ │ └── Agradecimientos.tex │ │ ├── Prefacio/ │ │ │ └── Prefacio.tex │ │ └── Capitulo X/ │ │ └── CapituloX.tex ├── settings/ │ └── fncystyle.sty ├── Libro de Ecuaciones Diferenciales.tex └── README.md


## Archivos Principales

- **`Libro de Ecuaciones Diferenciales.tex`**: Archivo principal que compila el libro completo.
- **`Cuerpo/Capitulos/`**: Carpeta que contiene los capítulos y secciones del libro.
  - **`Inicio/inicio.tex`**: Introducción al libro.
  - **`Dedicatoria/Dedicatoria.tex`**: Página de dedicatoria.
  - **`Agradecimiento/Agradecimientos.tex`**: Página de agradecimientos.
  - **`Prefacio/Prefacio.tex`**: Prefacio del libro.
  - **`Capitulo X/CapituloX.tex`**: Capítulos individuales del libro.
- **`settings/fncystyle.sty`**: Archivo de estilo para personalizar los capítulos.

## Requisitos

Para compilar este proyecto, necesitas tener instalado un entorno LaTeX. Se recomienda usar **TeX Live**, **MikTeX** o **Overleaf**.

## Compilación

1. Abre el archivo `Libro de Ecuaciones Diferenciales.tex` en tu editor LaTeX.
2. Compila el archivo usando `pdflatex`, `xelatex` o `lualatex` según tu preferencia.
3. El archivo PDF generado contendrá el libro completo.

## Personalización

- Puedes modificar los capítulos y secciones editando los archivos correspondientes en la carpeta `Cuerpo/Capitulos/`.
- El estilo de los capítulos puede ajustarse en el archivo `settings/fncystyle.sty`.
